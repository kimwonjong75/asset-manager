# 조건부 채널 KR-size prelock 결과

실행일: 2026-07-16  
가설: 대형주(A)는 20일 돌파가 55일보다 유리하고, 기타주(B)는 55일 돌파가 20일보다 유리하다.  
범위: 개발 2010-2019, 검증 2020-2022. 잠금표본 2023-2025는 열지 않았다.

## 결론

**검증구간의 방향성은 가설과 반대였다.** BASE 비용에서 `ΔA=-0.485R`, `ΔB=+0.211R`, interaction `I=ΔA-ΔB=-0.696R`이었다. 즉 대형주는 55일이 더 좋았고, 기타주는 20일이 더 좋았다. DOUBLE 비용에서도 `I=-0.899R`로 같은 반대 방향이었다.

다만 이 값은 ALL_20과 ALL_55에서 발생한 서로 다른 거래들의 그룹별 평균 R 차이인 비짝지은 기술통계다. 블록 부트스트랩 신뢰구간이 없고 G8·G11이 미완료이므로 **최종 기각이나 채택 판정이 아니라, 현재 가설을 지지하지 않는 prelock 증거**로 해석한다.

## 데이터 게이트

- 통과: G1~G7, G9, G10
- G7: 가격 단절 없는 발행주식수 변화 1,639건을 가격 비조정 사건으로 분리했다. 남은 미분류 사건 185건, 영향 종목 168개는 성과 확인 전에 전 기간 투자대상에서 제외했고 월말 스냅샷을 전수검증했다.
- 미완료(lockbox-only): G8 합병대가 테이블, G11 data.go.kr KRX 공식 10표본 교차검증
- 3,447종목, 156개월을 실제 OHLCV 엔진으로 처리했다.

## 해석상 제한

1. G8 미완료 동안 조기 종료 시계열은 최종 실거래 종가로 임시 청산한다. 합병의 실제 경제적 대가가 반영되지 않았으므로 prelock 탐색용이다.
2. G11 공식 교차검증이 없어 lockbox를 열 수 없다.
3. 직접 추정량의 짝지은 구성과 60거래일 블록 부트스트랩 10,000회가 아직 연결되지 않았다.
4. 모든 전략의 절대성과가 매우 낮다. 따라서 가설 비교와 별개로 비용·포지션 원장·상장종료 처리의 독립 검산을 한 번 더 수행해야 한다.

## 권고

현재 상태에서 lockbox를 열지 않는다. 다음 순서는 (1) 거래 원장 표본 검산, (2) 짝지은 ΔA/ΔB와 블록 부트스트랩 연결, (3) G8 합병대가 구축, (4) G11 공식 10표본 확인이다. 이 네 단계가 끝난 뒤에도 검증구간 interaction이 음수라면 원 가설은 기각하고 `REVERSE`를 새 가설로 재등록해야 한다.
