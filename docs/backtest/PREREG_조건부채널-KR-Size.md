# 사전등록 — 조건부 채널 (KR 대형주 축) `conditional-channel-kr-size-v1`

## §1. 목적과 범위

원래 가설(`conditional-channel-v1`, `PREREG_조건부채널검증.md`)에서 한국 대형주 축만 분리한 파생 가설이다.

**분리 이유**: WICS 업종 데이터 없이도 marcap의 point-in-time 시가총액만으로 대형주(상위 20%) 판정이 가능하므로, leader/업종 대장주 축을 제외하고 KR 단독으로 먼저 검증 가능한 범위를 확정한다.

**이 문서의 역할**: 결과를 보기 전에 방법론을 동결한다. 어떤 데이터도 시뮬레이션하기 전에 작성하였다.

---

## §2. 가설 정의

> **"KR KOSPI·KOSDAQ 보통주 중 시가총액 상위 20%(대형주)는 20일 신고가 돌파 진입이 유리하고, 그 밖의 종목은 55일 신고가 돌파 진입이 유리한가?"**

- **주 추정량 I** = ΔA − ΔB  
  - ΔA = mean(R20,A − R55,A): 대형주 그룹에서 20일채널 수익 − 55일채널 수익  
  - ΔB = mean(R20,B − R55,B): 비대형주 그룹에서 동일  
- **예상 방향**: ΔA > 0 (대형주는 20일이 유리), ΔB < 0 (비대형주는 55일이 유리), I > 0

**등록 조건**: 어떤 성과·수익률·돌파 신호도 이 시점에 계산된 적 없음.  
**데이터 상태**: `KR_SIZE_FEASIBLE_PENDING_IMPLEMENTATION` — marcap 데이터 가용성 확인 완료(DATA_FEASIBILITY_KR.md), 재현 가능 파이프라인은 이후 구현.

### 4.2 채널 파라미터
- 진입 룩백: 20일(단기), 55일(장기)
- 청산 룩백: 20일 고정
- 신호: `close[t] > max(high[t-lookback..t-1])` 엄격 부등호(현재 바 제외)
- 공통 시작: 55선행바 + ATR20 준비 후

### 4.3 그룹 정의
- **그룹 A**: 월말 시가총액(종가 × 발행주식수) 상위 20% (80백분위 이상)
- **그룹 B**: 나머지 분류 가능 보통주 (A의 정확한 여집합)
- **제외**: 시가총액 해석 불가 종목 → `unclassifiable=true`, B로 흡수 금지
- **tie-break**: 80백분위 동률은 종목코드(securityId) 오름차순

### 4.4 표본 분할
| 구간 | 기간 | 규칙 |
|------|------|------|
| **개발** | **2010-01-01 ~ 2019-12-31** | 최적화 금지(코드·데이터 오류 확인만) |
| **검증** | **2020-01-01 ~ 2022-12-31** | 개발과 완전 분리된 홀드아웃 (겹침 없음) |
| **잠금 표본** | **2023-01-01 ~ 2025-12-31** | **SEALED PSEUDO-OOS — 공식 교차검증 완료 전 미개봉** |
| 전향적 표본 | 2026-07-16 이후 | 현재 존재하지 않음 |

- 개발(2010-2019) + 검증(2020-2022) + 잠금(2023-2025): 세 구간 완전 비중첩

- **잠금 개봉 조건**: data.go.kr 서비스키로 KRX 공식 데이터 교차검증 완료 후

- 순열: 1,000회, 시드 20260716, 시장·월별 A비중 보존
- 다중 비교: Holm 보정
- 구간: 양측 95% CI

### 4.7 채택 기준 (KR 단독)
아래를 모두 만족해야 `SUPPORTED_FOR_PAPER_PILOT`:

1. 잠금 KR I의 95% CI 하한 > 0
2. 점추정 ΔA > 0, ΔB < 0
3. ADAPTIVE가 ALL_20·ALL_55 각각보다 CAGR +1.0%p, Sharpe +0.10 이상
4. ADAPTIVE MDD가 더 나은 기준선 대비 2.0%p 초과 악화 없음
5. 비용 2배에서도 I > 0, ADAPTIVE 열등 아님
6. A·B 각 30종목+, 잠금 100청산거래+
7. 최대기여 종목·연도 제거해도 I 부호 양수 유지
8. ADAPTIVE > REVERSE, 무작위 라벨 95백분위 초과
9. 룩어헤드·생존편향·기업행위·중복바·통화오류 미발견(QA)
