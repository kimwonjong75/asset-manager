# 사전등록 — 조건부 채널 (KR 대형주 축, 역방향) `conditional-channel-kr-size-reverse-v1`

> ⚠ **이 가설은 검증되지 않은 후속 가설이다.**  
> 잠금표본(2023-2025)을 한 번도 열지 않은 상태에서 2026-07-16 사전등록됐다.  
> G8(합병대가 테이블)·G11(KRX 공식 교차검증) 완료 전까지 잠금표본을 열지 않는다.

---

## §1. 등록 배경

`conditional-channel-kr-size-v1`의 prelock 탐색 단계(개발 2010-2019, 검증 2020-2022)에서 다음이 관측됐다.

| 항목 | 값 |
|------|-----|
| 원 가설 예상 방향 | I > 0 (대형주→20일, 비대형주→55일) |
| 검증구간 BASE I | **−0.696 R** |
| 검증구간 DOUBLE I | **−0.899 R** |
| 부트스트랩 월별 I 점추정 | −0.213 R |
| 부트스트랩 95% CI | [−1.228, 0.975] |
| 엔진 검산 결과 | rollingPriorExtreme 불일치 0건, 48거래 오류 0건, 불변식 15/15 통과 |
| 감사 최종 판정 | **ORIGINAL_NOT_SUPPORTED_REVERSE_PREREGISTERED** |
| 감사 보고서 | docs/backtest/LEDGER_AUDIT_KR_SIZE.md |

두 비용 조건(BASE/DOUBLE)의 직접추정량이 모두 음수이고 엔진 버그가 발견되지 않았으므로, prelock 설계(PRELOCK_RESULTS_KR_SIZE.md §7 권고)에 따라 역방향 가설을 새로 사전등록한다.

**명시적 경고**: 이 사전등록은 lockbox를 여는 행위가 아니다. 검증구간 결과가 새 가설을 선호할 가능성을 높이지만, 잠금표본을 열기 전까지는 채택 또는 기각 판정을 내릴 수 없다.

---

## §2. 가설 정의

> **"KR KOSPI·KOSDAQ 보통주 중 시가총액 상위 20%(대형주)는 55일 신고가 돌파 진입이 유리하고, 그 밖의 종목은 20일 신고가 돌파 진입이 유리한가?"**

- **주 추정량 I** = ΔA − ΔB  
  - ΔA = mean(R20,A − R55,A): 대형주 그룹에서 20일채널 수익 − 55일채널 수익  
  - ΔB = mean(R20,B − R55,B): 비대형주 그룹에서 동일  
- **예상 방향**: ΔA < 0 (대형주는 55일이 유리), ΔB > 0 (비대형주는 20일이 유리), **I < 0**  
- 원 가설과 동일한 측정량 I를 사용하되 부호 방향이 반전된다.

---

## §3. 원 가설과의 차이

| 항목 | `conditional-channel-kr-size-v1` | `conditional-channel-kr-size-reverse-v1` |
|------|----------------------------------|------------------------------------------|
| 예상 방향 | I > 0 | **I < 0** |
| 주 전략 | ADAPTIVE (A→20, B→55) | **REVERSE (A→55, B→20)** |
| 대조 전략 | REVERSE (A→55, B→20) | ADAPTIVE (A→20, B→55) |
| 등록 시점 상태 | 어떤 데이터도 없었음 | 개발·검증 결과를 본 뒤 역방향 등록 |
| 잠금표본 상태 | 미개봉 | **미개봉 (동일)** |

우주, 파라미터(ATR 20, Stop 2×ATR, 진입 필 delay 1일, 위험한도 0.5/12/25%), 비용 모형, 블록 부트스트랩(60거래일 블록·10,000회·시드 20260716)은 원 가설과 동일하다.

## §4. 방법론 동결 파라미터

### 4.1 설정 파일

```
scripts/backtest/conditionalChannel/preregistered-config-kr-size-reverse-v1.json
```

**등록 시각**: 2026-07-16 (한국 시간)  
**등록 조건**: 개발·검증구간 결과를 본 뒤 등록; 잠금표본(2023-2025)은 여전히 미개봉.  
**데이터 상태**: 잠금 미개봉. G8·G11 미완료.

### 4.2 채널 파라미터

원 가설과 동일 (lookback값 자체 변경 없음, 그룹별 배정이 반전됨):
- 진입 룩백: 20일(단기), 55일(장기)
- 청산 룩백: 20일 고정
- 신호: `close[t] > max(high[t-lookback..t-1])` 엄격 부등호(현재 바 제외)
- 공통 시작: 55선행바 + ATR20 준비 후

### 4.3 그룹 정의

원 가설과 동일:
- **그룹 A**: 월말 시가총액(종가 × 발행주식수) 상위 20% (80백분위 이상)
- **그룹 B**: 나머지 분류 가능 보통주 (A의 정확한 여집합)
- **제외**: 시가총액 해석 불가 종목 → `unclassifiable=true`, B로 흡수 금지

### 4.4 표본 분할 (원 가설과 동일)

| 구간 | 기간 | 상태 |
|------|------|------|
| **개발** | 2010-01-01 ~ 2019-12-31 | 결과 이미 공개됨 |
| **검증** | 2020-01-01 ~ 2022-12-31 | 결과 이미 공개됨 |
| **잠금 표본** | 2023-01-01 ~ 2025-12-31 | **SEALED — G8·G11 완료 전 미개봉** |
| 전향적 표본 | 2026-07-16 이후 | 현재 존재하지 않음 |

### 4.5 비용 모형

원 가설과 동일 (변경 없음, 역방향 가설이라도 비용 설정을 유리하게 변경하지 않는다):

| 기간 | 세율(bps) | 근거 |
|------|-----------|------|

### 4.6 통계 설계

원 가설과 동일:
- 블록 부트스트랩: 60거래일 블록, 10,000회, 시드 20260716
- 순열: 1,000회, 시드 20260716, 시장·월별 A비중 보존
- 다중 비교: Holm 보정
- 구간: 양측 95% CI

### 4.7 채택 기준 (역방향)

아래를 **모두** 만족해야 `REVERSE_SUPPORTED_FOR_PAPER_PILOT`:

1. 잠금표본 I의 95% CI **상한 < 0**
2. 점추정 ΔA < 0 (대형주는 55일이 유리), ΔB > 0 (비대형주는 20일이 유리)
3. REVERSE가 ALL_20·ALL_55 각각보다 CAGR +1.0%p, Sharpe +0.10 이상
4. REVERSE MDD가 더 나은 기준선 대비 2.0%p 초과 악화 없음
5. 비용 2배(DOUBLE)에서도 I < 0, REVERSE 열등 아님
6. A·B 각 30종목+, 잠금 100청산거래+
7. 최대기여 종목·연도 제거해도 I 부호 음수 유지
8. REVERSE > ADAPTIVE, 무작위 라벨 5백분위 미만(I < 0 방향으로)
9. 룩어헤드·생존편향·기업행위·중복바·통화오류 미발견(QA)

## §5. 잠금표본 개봉 전제 조건

잠금표본을 열기 전 완료해야 하는 게이트:

| 게이트 | 상태 | 설명 |
|--------|------|------|
| G1~G7, G9, G10 | ✅ 완료 | 원 가설 prelock 단계에서 통과 |
| **G8** | ❌ 미완료 | 합병대가 테이블(merger_proceeds.json) 구축 필요 |

G8과 G11이 모두 통과될 때까지 잠금표본을 열지 않는다. 게이트 미완료 상태에서 잠금 실행 시 엔진이 `WAITING_FOR_MERGER_PROCEEDS` / `WAITING_FOR_KRX_CROSSCHECK` 오류로 종료된다.
