# Phase 2 사전등록 — Global 후보 신호의 "엄격한 전략 검증"

**작성 시점: 2026-07-13. 데이터로 성과를 보기 전에 규칙을 고정한다.**
이것은 "검증된 돈 버는 전략의 실행"이 아니라 **"Global 후보 신호가 실제 거래 규칙·비용·아웃오브샘플에서도 버티는가"의 검증**이다.

## 0. 전제와 경계 (과신 금지)
- Phase 1 결과: 순환은 실재(전 트랙). US 섹터 모멘텀=깨끗한 귀무(예측 불가). **Global(국가/자산)은 모멘텀이 정보 보유**(무작위 대비 백분위 97~100%), 사전등록 규칙상 **h=3/6에서 통과**(교정 후). 단 h=1 통과는 사라졌고 표본이 짧아(2011-03~, 184개월) **fragile**.
- **중요**: Phase 1의 h=3/6은 "미래 3/6개월 초과수익 차이"라는 **평가 지표**일 뿐, "3/6개월 보유 전략이 정답"이라는 뜻이 **아니다**. 보유기간·리밸런싱·Top N·현금 규칙은 **여기서 새로 고정**한다(Phase 1에서 유도된 값이 아님).
- US 섹터: 단순 모멘텀 전략으로는 **Phase 2 제외 유지**. KR 섹터: **보조 관찰**로만(저검정력, 결론 근거 아님).

## 1. 유니버스 (고정)
- **SPY, EWY, EWJ, MCHI, GLD, DBC** — 6종.
- 데이터: Yahoo v8 수정(총수익) `adjClose` (`sectorRotation/cache/`). 갭 절단 적용. **공통구간 2011-03~ 마지막 완료월(현재 2026-06)**. 미완료 월 제외.
- 상장 전·결측 구간은 미조정 대체 금지(Phase 1과 동일 원칙).

## 2. 신호 (Phase 1과 동일 — 새 튜닝 금지)
- 시점 T(월말)의 모멘텀 점수 = `(rel_3 + rel_6 + rel_12)/3`, `rel_k(i)=R_k(i)-mean_j R_k(j)` (Phase 1 §3 동일 구현 재사용).

## 3. 1차 전략 (완전 사전명세·튜닝 금지)
아래 **단일 구성**을 대표 전략으로 고정한다. 결과가 나빠도 바꾸지 않는다.
- **선정**: 매월 말 점수 상위 **N=2** 종목 동일가중.
- **리밸런싱**: **월간**. 월말 T 신호 → **익월 첫 거래일 시가 체결**(룩어헤드 금지).
- **회전 완충(buffer)**: 보유 종목이 상위 **3위(상위 절반) 이내**면 유지, 밖으로 나가면 신규 상위로 교체(불필요 회전 억제).
- **절대 모멘텀(현금 대피)**: 선정된 종목이라도 자신의 **12개월 절대 총수익 ≤ 현금(BIL) 동기간 수익**이면 그 슬롯을 **현금(BIL)**으로 대체(듀얼 모멘텀). 6종 중 강제 상시보유 금지.
- **현금 자산**: BIL(2007 상장, 2011~ 구간 내내 존재)의 총수익. 교차검산으로 FRED DTB3 합성 현금 병기.

## 4. Phase 1이 정해주지 않은 파라미터 — 여기서 고정 (Codex 지적 반영)
- 보유기간/리밸런싱 = 월간(위). Top N = 2. 완충 = 상위 3위. 현금규칙 = 듀얼모멘텀 vs BIL.
- 이 값들은 **관례적 기본값**으로 사전 선언하며, 이 데이터에 맞춰 고른 것이 아니다. §7 민감도는 **보고만** 하고 최적값 선택에 쓰지 않는다.

## 5. 거래비용 (고정)
- 기준: **편도 0.10%**(왕복 0.20%). 민감도 스윕 {0, 0.05, 0.10, 0.20, 0.25%} — 보고용, 선택용 아님.
- 회전율 기반으로 비용 차감. 슬리피지는 편도 비용에 포함 가정.

## 6. 벤치마크 (고정)
- SPY 매수후보유 / 6종 동일가중 매수후보유 / 60·40(SPY·AGG) / 현금(BIL).

## 7. 검증 설계 (짧은 표본 대응)
- 전략은 자유 파라미터가 없도록 §3에서 완전 명세 → **전 구간** 성과 + **고정 시기 분할**(전반 2011-03~2018-12 / 후반 2019-01~현재)로 안정성 확인.
- **확장 워크포워드 자산곡선**도 제시(파라미터 선택용이 아니라 시점별 견고성 시각화용).
- **민감도 그리드**: N∈{1,2,3} × 리밸런싱∈{월,분기}. **전부 표로 보고**하되 **"최고 셀 선택 금지"** — 1차 전략은 §3 단일 구성. 그리드는 "인접값에서 급붕괴 여부"만 본다.
- **h=3/6은 탐색적 후보로 표기** — Phase 2가 실제 거래·비용에서 재확인하는 대상이지 확정 근거가 아님.

## 8. 성과 지표 (고정)
- 전략: CAGR / MDD / Sharpe / Sortino / Calmar / 연도별 수익 / 최악 1년·3년.
- 운용현실: 연회전율 / 총비용 차감폭 / 시장 노출도 / 상승·하락·급반등·2022·코로나 국면별.
- 강건성: **단일 자산이 초과수익의 몇 %를 만들었나** / **상위 5개 거래 제거 후 성과** / 비용 0.3%에서 장점 잔존 여부.

## 9. 통화 (Codex 항목 7)
- **1차 = USD**(모든 자산 USD 표시라 순수 전략 성과). **보조 = KRW 환산**(USD/KRW 히스토리로 환산, KRW CAGR·MDD 별도 리포트). 환헤지는 별도 실험(범위 밖).

## 10. 판정 규칙 (사전 선언)
- "통과 후보"로 보려면: 비용 반영 후 SPY·동일가중 대비 우수 **AND** 시기 분할 양쪽에서 방향 유지 **AND** 단일 자산/상위 거래 제거에도 결론 유지 **AND** 편도 0.3%에서도 장점 잔존.
- 하나라도 무너지면 "Global 순환 신호는 관찰되나 거래전략으로는 불충분"으로 정직히 기록. 결과와 무관하게 그대로 보고(튜닝 재탐색 금지).
- KR 보조 전략은 동일 규칙으로 돌려 **참고 표**로만 첨부(저검정력 명시).

## 11. 룩어헤드·무결성 가드
- 신호=월말 T 종가까지만. 체결=익월 첫 거래일. 미래 종가는 성과 측정에만.
- 상장 전 소급·미조정 대체·미완료월 포함 금지. 시드 고정(재현성). 합성 데이터 파리티로 체결·비용·회전 로직 골든값 고정.
